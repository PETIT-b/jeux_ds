Foire aux Questions


  • Qu'est-ce que 'Jeux DS'?
    'Jeux DS' est un projet dont le but est de référencer des informations sur les jeux DS et les consoles de la famille DS (DS, DSi,...)
  • Pourquoi avoir créé 'Jeux DS'?
    Ayant déjà envie de coder ce site avant, une opportunité de mêler (ré)apprentissage et plaisir s'est présentée : un projet scolaire devant remplir plusieurs conditions (flexbox, JS,...).
  • Comment est "codé" 'Jeux DS'?
    Jeux DS est codé en HTML, CSS et JavaScript. La partie HTML/CSS est codée en Flexbox. La partie JavaScript s'occupe de charger les entêtes, menu de Navigation et pied de pages (un peu comme les views dans les frameworks). Elle s'occupe aussi de récupérer les données dans un JSON et de les "injecter" dans le code HTML.
  • Où puis-je trouver le code de 'Jeux DS'?
    Actuellement, quasi la totalité du code est sur codepen.io, et divisé en plusieurs parties :
    • Page d'accueil
    • Page des DS originales
    • Page des Firmwares de la DSi
    • Page de la FAQ
    • Code JS pour les fonctions qui chargent les menus, entêtes,...
    • Code JS pour 'stocker' le JSON, extraire les données,...
